# 🏠 我们的家 - 家庭网站

> 宋世伟 & 华宇 的温暖小窝

## 功能

- 📸 **家庭相册** — 上传、分类、预览照片
- 📝 **家庭日记** — 记录每一天的小确幸
- 📋 **备忘录** — 购物、待办、提醒、账单
- 🍳 **家庭菜单** — 本周/下周菜单规划

## 技术栈

- 纯 HTML + CSS + JavaScript（无框架依赖）
- LocalStorage 数据持久化
- 响应式设计（手机 & 电脑）

## 部署

已部署至 GitHub Pages：
https://TrWestdoor.github.io/family-website

## 本地预览

```bash
# 方式1：直接用浏览器打开 index.html

# 方式2：用 Python 临时服务器
cd /tmp/family-website
python3 -m http.server 8080
# 访问 http://localhost:8080
```
